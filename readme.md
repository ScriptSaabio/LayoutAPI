# 🧙‍♂️ Harry Potter API

![API](https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcQeCbVdCh7A0sAjv3UF_7s6IHDhximRQkIfEA&s)

Este projeto consome a API do universo Harry Potter para exibir informações sobre bruxos, suas casas e outros dados do mundo mágico.

---

## ✨ Funcionalidades

- Busca de personagens do universo Harry Potter
- Exibição de informações como nome, casa e ator
- Interface dinâmica e interativa
- Integração com API externa

---

## 🧙 Bruxos (Contribuidores)

- [@ScriptSaabio](https://github.com/ScriptSaabio)

---

## 🛠 Tecnologias utilizadas

- HTML5  
- Tailwind CSS  
- JavaScript (Vanilla)

---

## 📡 API utilizada

Projeto baseado na Harry Potter API (exemplo: https://hp-api.onrender.com/)

---

## 🚀 Objetivo

Praticar consumo de APIs, manipulação do DOM e construção de interfaces dinâmicas com JavaScript.
